Integration Guide

Civica CCM Integration — Scope & Coverage

Everything your college needs to know about how Admissions+ connects with Civica CCM, what data is supported, and how it flows.

How the integration works

Admissions+ connects to your Civica CCM system through the Applicaa Civica Integration App, a lightweight Windows application installed on your college network. The app securely bridges Admissions+ and your Civica CCM system, transferring applicant and student data using Civica's REST Open API.

Civica CCM Records created & updated

Safe by design: Admissions+ only ever creates new records or updates existing values in Civica CCM — it never deletes or blanks data that is already there. Any data removal is always carried out in Civica CCM by your college staff.

SECTION 7

Lookup data (reference lists)

Admissions+ imports your Civica CCM reference lists so that dropdowns and selection fields match your MIS exactly. This keeps data consistent and avoids validation errors during transfer.

SECTION 8

Setup requirements

What you need from Civica CCM

  • Open APIs enabled — your Civica CCM system must have Open APIs enabled in College Management
  • Integration agent account — a dedicated agent email and password (not a personal login)
  • Five API endpoint URLs — Account, Reference Data, Application, Course, and Utilities API base URLs for your college

What you need from Admissions+

  • Admissions+ login — email, password, and your college subdomain
  • Integration App installer — downloaded from the Admissions+ admin panel (Systems > MIS Apps)

IT requirements

  • Windows desktop — the Integration App runs on Windows
  • Network access — whitelist *.applicaa.com, api.applicaa.com, and admissionsuk.s3.eu-west-2.amazonaws.com
  • Proxy — if applicable, whitelist both Applicaa Civica Integration App.exe and Updater.exe

How syncs work

  • Data transfers are initiated manually from the Integration App by staff
  • Lookup data refreshes automatically each time the app is launched
  • The app auto-updates on each launch to stay current
  • Connection status for both Admissions+ and Civica CCM is shown in the app
